Projects

As part of my four year college programme I have developed
a number of projects which are detailed below.

Inventory Management System

Collaborated with three others and developed an Inventory Management System using C#. The system was responsible for stock taking, user management and trail auditing.

Smart Access Control Monitoring Sytem

Spearheaded the development of a Smart Acess Control and Moitoring System using Proteus and MpLab. System contained a pin code digital door lock, smoke detector and siren.

Biometric based Miners Check-in system

As part of my internship project, I developed a biometric based miners check in system that allowed miners to check in for work using their fingerprints.

Mobile self checkout using Mask RCNN and Atrous Convolution

For my final year project I developed a mobile based self checkout system tha uses Mask RCNN and Atrous Convolution. This system allows users to scan their products using phone camera, edit cart and check out of a supermarket via mobile phones beating the need for tills.

Web based Internal Directory

Removed the need for yellow pages by developing a web hosted internal directory that shows the company landline numbers.
